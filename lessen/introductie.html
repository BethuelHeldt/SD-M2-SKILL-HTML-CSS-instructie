Introductie basiscursus HTML/CSS

In deze lessen serie leer je de programmeertalen HTML (Hyper Text Markup Language) en CSS (Cascading Style Sheets). Deze talen vormen de basis van bijna elke website. Je leert Semantische HTML. Lees hier over Semantische HTML.

Opdrachten

Je maakt deze mudule 2 opdrachten.

HTML/CSS is behaald als beide opdrachten (blog en landingspage) zijn behaald met minimaal een voldoende.

Blog

Les 1t/m 5
Je maakt een blog-site met navigatie en verschillende artikelen. De HTML is semantisch en de site is responsive.

Beoordeling:
De blog is gemaakt tot en met les 5. Er is een homepage met links naar drie artikelen waarvan twee artikelen zelf gemaakt zijn. Alles is responsive vanaf 400px en alle code is gecontroleerd in de W3C Validator.

Landingspage

Les 6 t/m 7
Je maakt een eenvoudige landingspage met image slider. De HTML is semantisch en de site is responsive.

Beoordeling:
Alle video's van de landingspagina zijn gekeken en nagebouwd. De landingspage is aangevuld met een eigen sectie met content en de imageslider is uitgebreid. Alle code is volledig semantisch en foutloos, te controleren door de W3C Validator. De gehele landingspage is responsive vanaf 400px.
